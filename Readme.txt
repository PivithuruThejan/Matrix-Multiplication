1. How to run the Matrix Multiplication for Sequential Implementation

* Go to the file location which contain the cpp files
* Open a terminal

#..............For Linux machines..............................

* Enter the command "g++  Sequential.cpp -o sequential" to compile
* Enter the command "./sequential" to run the program
* Then enter the matrix dimention and number of rounds


#..............For Windows machines..............................

* Enter the command "g++  Sequential.cpp -o sequential" to compile
* Enter the command "sequential" to run the program
* Then enter the matrix dimention and number of rounds

=================================================================
